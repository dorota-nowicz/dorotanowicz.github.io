NY Taxi Demand Forecast

In this project ,the taxi demand in NYC by zones is predicted. The dataset is taken from the NYC government website and corresponds to 2020 and 2021 years. The Project passes through data pre-processing and analysis and then use some machine learning regression models like Linear Regression, Random Forest and Gradient Boosting. The dataset is splitted into train and test sets by TimeSeriesSplit() class from Scikit-learn. After analysing errors of each model, a model which returns the lowest Root Mean Squared Error (RMSE) is selected and checked on the test data (data not seen during model fitting nor validation). Finally, the short resume with a report example for stakeholders is generated.
