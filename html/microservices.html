Microservices

This project consisted of the development of several web applications to offer services focused on the management of water supply and sewage treatment. The applications allow among other functionalities:

  • Receive and send data by the operators
  • Review the inventory of the work carried out by maintenance
  • Create parts associated with the control and maintenance of network elements
  • Generate alerts and warnings based on the measurements provided by the control devices

The collected information gives technicians the ability to quickly make smart decisions, optimise operator jobs, and minimise costs.

It is worth mentioning that this Client has its own information systems and a centralised database where it stores data collected by its remote control. Integration with the applications is done through Open API (Swagger). OpenAPI is a selection of machine-readable interface files for describing, producing, consuming, and displaying web services.

Swagger is a set of concrete tools to work with the OpenAPI format. Making the applications to access some values of the Client database, a swagger.json file is generated, which is used to generate a client whose objective is to respond to the requests of our applications and thus guarantee the connection with the external database.
